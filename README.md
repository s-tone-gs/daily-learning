# daily-learning

日々Claude Codeを使って学んだことを、カテゴリごとにディレクトリ分けして保存しておくリポジトリです。

## 目的

学んだ知識を体系的に整理し、後から見返せるようにすること。Claude Codeとの学習結果をそのままMarkdownファイルとして残すことで、別のメモに転記する手間をなくします。

## 構成ルール

- カテゴリごとにディレクトリを作成します（学習内容が出てきたタイミングで適宜追加されます）
- 各カテゴリディレクトリの中に `YYYY-MM-DD-トピック名.md` という形式でファイルを保存します
- カテゴリ名やファイル名はClaude Codeが内容から判断して提案します

新しい分野を学ぶたびに、カテゴリディレクトリが自然と増えていきます。
